# 🛠️ Ultimate Windows Maintenance Suite v8.3 (IT Command Center)
**Created by:** ElJoo

A powerful, all-in-one Batch script designed specifically for IT Help Desk professionals and System Administrators. This script automates 70 of the most common Windows troubleshooting, maintenance, and network tasks, turning hours of repetitive work into a single click.

---

## 🚀 Key Features
The script is divided into 7 organized categories:

1. **System & Security:** Run SFC, DISM, Chkdsk, clear temp files, and manage Windows Defender.
2. **Network Troubleshooting:** Full network resets, DNS flushing, changing DNS to Google, and retrieving saved WiFi passwords.
3. **Power & Info:** Generate battery health reports, extract Windows/BIOS product keys, and view detailed PC hardware specs.
4. **Help Desk Tools:** Clear caches for Browsers, Teams (Classic & 2.0), and Outlook. Fix Windows Update issues and sync system time.
5. **BitLocker & Disk:** Extract BitLocker recovery keys, check disk SMART health, block/unblock USB ports, and safely enter Safe Mode.
6. **Final Polish:** Deep disk cleanup, clear display/font caches, enable hidden administrator accounts, and pause Windows updates.
7. **Expert Tools:** Rebuild icon caches, clear the clipboard, print network routes, and kill all "Not Responding" processes instantly.

---

## ⚙️ How to Use

1. **Download** the `IT_Command_Center_v8.3.bat` file.
2. **Run it:** Double-click the file. 
   *(Note: The script includes an auto-elevation feature. If you don't run it as Administrator manually, it will automatically prompt you for Admin rights to ensure all commands execute successfully).*
3. **Select a Command:** Type the ID number (1-70) of the task you want to perform and press `Enter`.
